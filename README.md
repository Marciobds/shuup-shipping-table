# Shuup Shipping Table
A shipping method add-on for Shuup based on price tables by region

## Compatibility
* Shuup v0.5.0
* [Tested on Python 2.7, 3.4 and 3.5](https://travis-ci.org/rockho-team/shuup-shipping-table)
